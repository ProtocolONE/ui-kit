# ui-kit
Collection of ui components

## Development

### Link ui-kit for development
In ui-kit folder:
```
yarn link
```

In your project folder:
```
yarn link ui-kit
```

### Usage
Get components as:
```
import { Button, SwitchBox } from 'ui-kit';
```

### Lints and fixes files
```
yarn lint
```
